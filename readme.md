# NeetCode Solutions 🚀

This repository contains my solutions to **NeetCode 150** problems, organized by topic.  
All solutions are written in **C++** and some are explained with a short approach.  

I also maintain **short notes and explanations** for each problem in this Google Doc:  
[My NeetCode Notes](https://docs.google.com/document/d/1JXj5SWAf7Gcw47UlEspkXIhnx6_Cba1Ad019LvNUxPk/edit?usp=sharing)

## 📂 Repository Structure

- `Arrays_and_Hashing/` – Problems related to arrays, hashmaps, and basic data structures  
- `Two_Pointers/` – Problems using two-pointer techniques  
- `Sliding_Window` - 
- `Stack/` – Stack-based problems  
- `Binary_Search/` – Problems solved using binary search  
- `Trees/` – Tree traversal and manipulation problems  
- `Graphs/` – Graph algorithms  
- `Dynamic_Programming/` – DP problems
- `Heap_Priority Queue/` - Heap problems

Each folder contains cpp files named after the problem with a link of the problem at the top.
